# Contributing

Thank you for contributing to Awesome Web Scraping 2026!

## How to Add a Tool

1. Fork this repository
2. Add your tool to the appropriate category in `README.md`
3. Use this format:

```
| [Tool Name](https://github.com/user/repo) | Stars | Short description (one sentence). |
```

4. Submit a pull request

## Requirements

- The tool must be **actively maintained** (updated within the last 12 months)
- Include a working link to the tool's homepage or repository
- Description should be **one sentence**, max 100 characters
- Star count should use `k+` format (e.g., `5k+`)
- Place the tool in the correct category

## Categories

If your tool doesn't fit an existing category, suggest a new one in your PR description.

## What We Accept

- Open-source scraping tools and frameworks
- Commercial tools with a free tier
- Free APIs that replace scraping
- Browser automation tools
- Proxy and anti-detection tools
- Data parsing libraries

## What We Don't Accept

- Tools that haven't been updated in 2+ years
- Duplicate entries
- Tools without documentation
- Paid-only tools with no free option
- Tools that violate terms of service of target websites
